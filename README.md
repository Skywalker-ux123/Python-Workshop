# 🚀 The Python Pipeline Workshop

Welcome to the companion code repository for our Python pipeline presentation! 
Here you will find starter templates to build your own Web Scrapers, AI Vision models, and Web Servers.

## 🛠️ Setup Instructions

1. **Clone this repository** to your local machine.
2. **Install the required libraries**. Open your terminal and run:
   ```bash
   pip install requests beautifulsoup4 selenium pandas matplotlib ultralytics opencv-python flask
3. Open the folders and look for files ending in _template.py.
4. Find the TODO comments and fill in the missing code! (If you get stuck, we have included the completed files for reference).

   Happy coding!
